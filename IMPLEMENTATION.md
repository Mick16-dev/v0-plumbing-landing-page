## Overview

Think of this as the behind-the-scenes map for the site.  
Each phase focuses on one part of the experience: first making it feel aligned with the brand, then making it easy to navigate, then building trust, and finally refining the little details that make it feel considered and human.

## Phase 1 – Foundations & Identity
- [x] **Typography**: Bricolage Grotesque for headings and Outfit for body text to create a friendly, confident voice.
- [x] **Color system**: Updated OKLCH palette for softer depth and warmer highlights, avoiding harsh contrast.
- [x] **Surfaces & texture**: Light grain, mesh gradients, and gentle glassmorphism to add warmth without feeling “techy”.
- [x] **Underlying setup**: Metadata, viewports, and local business SEO tuned so the site is technically solid.

## Phase 2 – Navigation & Hero
- [x] **Header**: Glass-style header with clear logo, simple navigation, and a primary “Need help now?” CTA.
- [x] **Hero flow**: 
    - [x] Step-based intro using Framer Motion, so visitors are guided rather than overwhelmed.
    - [x] Subtle “AI scanning” effect for uploads that feels reassuring instead of flashy.
    - [x] Friendly success state showing an estimate and next steps in plain language.
    - [x] Soft mesh gradient background for a calm, premium first impression.

## Phase 3 – Authority & Social Proof
- [x] **Trust strip**: Refined trust badges with gentle animation and clear labels (licenses, experience, response times).
- [x] **Service highlights**: Interactive pillar cards that explain benefits in everyday language, not technical jargon.
- [x] **Before/after gallery**: 
    - [x] Comparison sliders with smooth handles and clear captions.
    - [x] High-quality imagery that feels real and grounded.
- [x] **Stories & reviews**: 
    - [x] Testimonial layouts that read like real customer stories.
    - [x] Video / quote cards with subtle motion and unobtrusive social proof cues.

## Phase 4 – Conversion & Final Polish
- [x] **Pricing overview**: Simple tiers with one clearly highlighted “most common choice”, plus a clear next step button.
- [x] **Questions section (FAQ)**: Accordion styled to be easy to scan, with structured data added for answer engines.
- [x] **Footer**: Clear contact details, service areas, and legal links, framed by a soft gradient background.
- [x] **Lead capture modal**: Glass-styled modal with a short, step-by-step flow and a warm confirmation state.

## Phase 5 – Content & Assets
- [x] **Visual assets**: High-quality, realistic expert portraits and environment imagery.
- [x] **Copy**: Human, conversational tone in EN/DE focused on clarity, reassurance, and next steps.
- [x] **Performance**: Assets and dependencies tuned for fast load and strong Lighthouse scores.

## Phase 6 – Advanced Enhancements
- [x] **Micro-interactions**: Gentle magnetic pull or hover responses on primary CTAs to make them feel tactile.
- [x] **Scroll story**: “Sticky” stacked sections to give a sense of moving through the space of the service.
- [x] **Expert trust signals**: Profiles, certifications, and experience highlights that support E-E-A-T.
- [x] **Diagnostic visuals**: Blueprint-style animation that makes the technical side feel understandable, not intimidating.
- [x] **Search & structured data**: Expanded JSON-LD schema with glossary and local service entities for better AEO.
